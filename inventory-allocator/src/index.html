Deliverr recruiting exercise

This program computes the best way an order can be shipped given inventory across a set of warehouses:


Unit Test 1 - Happy case


Input: {"apple":5}, [{"name":"owd","inventory":{"apple":5}}]

Expected: [{ owd: { apple: 5 } }]


Unit Test 2 - Simple split across warehouses


Input: {"apple":10}, [{"name":"owd","inventory":{"apple":5}},{"name":"dm","inventory":{"apple":5}}]

Expected: [{ owd: { apple: 5 } }, { dm: { apple: 5 } }]


Unit Test 3 - Not enough inventory at first warehouse, goes to second (Shipping from one)


Input: {"apple":5,"banana":5,"orange":5}, [{"name":"owd","inventory":{"apple":5,"orange":5,"banana":2}},{"name":"dm","inventory":{"banana":5}}]

Expected: [{ owd: { apple: 5 } }, { dm: { banana: 5 } }, { owd: { orange: 5 } }]


Unit Test 4 - Complex split from warehouses


Input: {"apple":5,"banana":10,"orange":5}, [{"name":"owd","inventory":{"apple":5,"orange":3,"banana":1}},{"name":"dm","inventory":{"banana":9,"orange":6}}]

Expected: [{ owd: { apple: 5 } }, { owd: { banana: 1 } }, { dm: { banana: 9 } }, { dm: { orange: 5 } }]


Unit Test 5 - Complex split from warehouses AND not enough inventory at first/Second warehouse AND three warehouses


Input: {"apple":5,"banana":5,"orange":5,"pineapple":6}, [{"name":"owd","inventory":{"apple":5,"orange":5,"banana":2,"pineapple":2}},{"name":"dm","inventory":{"banana":5,"pineapple":2}},{"name":"cam","inventory":{"banana":5,"pineapple":2}}]

Expected: [{ owd: { apple: 5 } }, { dm: { banana: 5 } }, { owd: { orange: 5 } }, { owd: { pineapple: 2 } }, { dm: { pineapple: 2 } }, { cam: { pineapple: 2 } }]


Unit Test 6 - Simple not enough inventory


Input: {"apple":10}, [{"name":"owd","inventory":{"apple":4}}]

Expected: []


Unit Test 7 - Order not at any stores


Input: {"apple":10,"banana":10}, [{"name":"owd","inventory":{"pear":4}},{"name":"dm","inventory":{"orange":4}}]

Expected: []


Unit Test 8 - No warehouses


Input: {"apple":10,"banana":10}, []

Expected: []
